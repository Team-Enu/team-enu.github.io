Team Enu

NTTグループの有志によるCTFチームTeam Enuです。

CTFTimeはこちら：https://ctftime.org/team/4894/

活動内容

  • DEF CON CTFやSECCONなど、様々なCTFに参戦
  • 毎週金曜日の13:00〜17:30まで勉強会を実施
    • モクモク会や集合勉強会　等（出入り自由）

  • Cyber Apocalypse 2024: Hacker Royale 28位（国内1位）
  • DEF CON CTF 予選 2023 世界31位

  • SECCON CTF 2022 予選 35位（国内9位）
  • DEF CON CTF 予選 2022 世界28位
  • SECCON CTF 2019 -Domestic-（国内）3位
